# 🎓 UniAssist – Smart University Assistant

UniAssist is an AI-powered assistant designed to help students with university-related queries such as course recommendations, information retrieval, and guidance.

---

## 🚀 Features

* 🤖 NLP-based query understanding
* 📚 College/course recommendation system
* 🔍 Semantic search using embeddings
* ⚡ Fast similarity search using FAISS
* 🌐 Interactive UI with Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Sentence Transformers
* FAISS

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/itzdhruv11/UniAssist.git
cd UniAssist
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run ui_app.py
```
